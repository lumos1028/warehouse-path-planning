## 🎯 使用说明

### 启动应用

```bash
cd /Users/fanyue/code
conda activate base
python backend/app.py
```

然后打开浏览器访问：**http://127.0.0.1:5000**

### 功能演示

#### 单机器人模式（推荐新手）
1. 点击"单机器人"按钮（已默认选中）
2. 在地图上点击一个位置设置**起点**（显示为绿色 S）
3. 再点击另一个位置设置**终点**（显示为红色 G）
4. 点击"**开始规划**"按钮
5. 查看执行结果：
   - 青色路线 = 最短路径
   - 右侧面板显示路径长度和坐标

**示例**：从左上角 (0,0) 规划到右下角 (49,49)

#### 多机器人模式（高级演示）
1. 点击"多机器人"按钮
2. 设置机器人数量（1-10个）
3. 点击"**生成随机任务**"
   - 系统为每个机器人随机分配起点和终点
   - 左侧面板列出所有任务
4. 点击"**规划所有路径**"
   - 为每个机器人规划最短路径
   - 网格显示所有路径
   - 统计总成功数和步数

- 尝试刷新页面或清除缓存

- 这是正常现象，取决于地图复杂度
- 50x50 地图通常 < 100ms

- 检查起点/终点是否在障碍上
- 检查是否存在路径（有时被障碍完全围困）

## 📚 核心算法说明

### A* 寻路算法
- **启发式函数**: 曼哈顿距离
- **评估函数**: f(n) = g(n) + h(n)
  - g(n) = 从起点到当前点的实际距离
  - h(n) = 从当前点到终点的估计距离
- **保证**: 在无权图上找到最优路径

### 时间复杂度
- 最坏情况: O(b^d)，其中 b 是平均分支数，d 是深度
- 实际通常是多项式时间

### 空间复杂度
- O(n) 其中 n 是网格中的节点数
